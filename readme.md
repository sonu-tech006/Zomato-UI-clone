# 🍔 Zomato UI Clone

🚀 A modern and responsive frontend clone inspired by the popular food delivery platform **Zomato**.
This project focuses on delivering a clean UI/UX experience with smooth navigation and real-world design structure.

---

## 🌐 Live Demo

🔗 https://zomato-ui-clone-kvsa0i0tr-sonu-tech006s-projects.vercel.app/

---

## 📌 Features

* 🔍 Search restaurants, cuisines, and dishes
* 📍 Location-based UI (e.g., Kanpur selection)
* 🍽️ Restaurant cards with images and details
* 📱 Fully responsive design (mobile + desktop)
* 🎨 Clean and modern UI inspired by real-world product
* ⚡ Smooth user experience and layout

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🔺 Next.js (App Router)
* 🎨 Tailwind CSS
* 🌐 Vercel (Deployment)
